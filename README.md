## Функціонал

### Основний (Варіант 33)
- **Меню з категоріями** — бургери, піца, суші, супи, салати, десерти, напої
- **Картка страви** — фото (emoji), назва, опис, ціна, рейтинг, час доставки, калорії
- **Кошик** — додавання/видалення товарів, зміна кількості, підрахунок суми
- **Форма замовлення** — адреса, телефон, спосіб оплати (готівка/картка/Apple Pay/Google Pay)
- **Екран підтвердження** — номер замовлення, час доставки, живий трекер статусу

### Додатковий функціонал (від розробника)
- **Темна/світла тема** — перемикач у шапці, збереження в AsyncStorage
- **Обране** — збереження страв, персистентність через AsyncStorage
- **Пошук** — по назві, опису та інгредієнтах
- **Сортування** — за популярністю, ціною (↑↓), рейтингом
- **Перегляд** — переключення між сіткою (2 колонки) та списком
- **Промокоди** — СМАЧНО20 (-20%), ПЕРШИЙ (-50 грн), ЛІТО10 (-10%)
- **Безкоштовна доставка** — прогрес-бар до порогу 500 грн
- **Анімації** — кнопка "Додати", серце в обраному, підтвердження замовлення
- **Live трекер замовлення** — анімований прогрес: прийнято → готується → їде → доставлено
- **Перегляд страви** — вибір кількості, пов'язані страви, інгредієнти

## Встановлення та запуск

### Вимоги
- Node.js ≥ 18
- npm або yarn
- Expo Go (на телефоні) або iOS Simulator / Android Emulator

### Кроки

```bash
# 1. Клонувати репозиторій
git clone <repo-url>
cd SmachnoApp

# 2. Встановити залежності
npm install

# 3. Запустити
npm start

# Відкрити на телефоні через Expo Go (QR-код)
# або:
npm run ios     # iOS Simulator
npm run android # Android Emulator
npm run web     # Браузер
```

## Архітектурні рішення

- **Context API + useReducer** для кошика — простіше Redux, але потужніше useState
- **AsyncStorage** для збереження теми та обраного між сесіями
- **useMemo** для оптимізації фільтрації та сортування меню
- **Animated API** для плавних анімацій без зовнішніх бібліотек
- **FlatList** з `numColumns` для адаптивного перемикання вигляду
- **LinearGradient** для виразних header-секцій
